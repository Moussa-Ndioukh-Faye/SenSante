# SenSante

Assistant de pre-diagnostic medical pour le Senegal

## Description

SenSante utilise le Machine learning pour aider au pre-diagnostic des maladies courant( paludisme, grippe, typhoide) a partir des symtomes du patient.

## Structure du projet 

- 'Data/' : donnees patients(CSV)
- 'models/' : Models ML serialise 
- 'api/' : API FastAPI
- 'frontend/' : Interface web
'notebooks/' : Scripts d'exploitation
